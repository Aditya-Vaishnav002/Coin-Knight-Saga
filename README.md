# Coin Knight Saga

Coin Knight Saga is a 2D platformer prototype developed using the Godot Engine.  
This project represents my first attempt at building a complete playable game while learning game development.

The game focuses on simple platforming mechanics, coin collection, and basic enemy interactions.

This project is currently in the **prototype phase**.

---

## Game Overview

In Coin Knight Saga, the player controls a knight exploring a colorful platforming world filled with coins, enemies, and environmental challenges.

The player must navigate platforms, avoid hazards, collect coins, and reach the end of the level.

The project serves as the **foundation for a larger game concept** that will later evolve into a darker narrative experience.

---

## Current Features

- Player movement and jumping mechanics
- Coin collection system
- Moving enemies
- Moving platforms
- Kill zones (player dies when touching enemies or hazards)
- Basic level progression
- Sound effects and background music
- Tutorial messages during gameplay
- Simple UI feedback

| Action | Key |

---

## Level Mechanics

The current prototype includes several gameplay elements:

### Platforming
The player must jump across platforms and avoid falling into dangerous areas.

### Enemy Interaction
Enemies patrol platforms.  
Touching an enemy currently results in player death.

### Moving Platforms
Some platforms move, requiring timing and careful jumping.

### Coin Collection
Coins are placed throughout the level to encourage exploration.

### Tutorial Guidance
In-game text provides hints to help new players understand the mechanics.
